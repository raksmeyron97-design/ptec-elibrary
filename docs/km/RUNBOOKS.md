# бһҹбҹҖбһңбһ—бҹ…бһҺбҹӮбһ“бһ¶бҹҶбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ (Operations Runbooks & Schedules)

_бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҗбҹ’бһ„бҹғбһ‘бһё бҹЎбҹў бһҒбҹӮбһҖбһҖбҹ’бһҖбһҠбһ¶ бһҶбҹ’бһ“бһ¶бҹҶбҹўбҹ бҹўбҹҰ (бһҖбһ¶бһҡбһ„бһ¶бһҡбһ‘бһё бҹҘ)бҹ” бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһҮбҹҶбһ“бһҪбһҹбһҹбҹҖбһңбһ—бҹ…бһҺбҹӮбһ“бһ¶бҹҶбһ…бһ¶бһҹбҹӢбҹ—бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `MONITORING.md`бҹ” бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ (Alerts) бһҠбҹӮбһӣбһҲбһ¶бһ“бһҠбһӣбҹӢбһҖбһ¶бһҡбһ”бһҫбһҖбһҹбҹҖбһңбһ—бҹ…бһҺбҹӮбһ“бһ¶бҹҶбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„ `ALERT-CATALOG.md`бҹ” бһўбҹ’бһ“бһҖбһҗбҹӮбһ‘бһ¶бҹҶбҹ– бһ”бҹ’бһҡбһ’бһ¶бһ“бһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡбһңбҹҒбһ” (Web-team lead)бҹ”_

## бһ•бҹ’бһ“бҹӮбһҖбһ‘бһё бҹЎбҹ– бһҖбһ¶бһӣбһңбһ·бһ—бһ¶бһӮбһҗбҹӮбһ‘бһ¶бҹҶ (Maintenance Schedules)

бһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢбһҖбһ»бҹҶбһІбҹ’бһҷбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ»бҹҮ Sev 1 / Sev 2бҹ”

### M1. бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғ (Daily) вҖ” бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ–бҹҒбһӣбһ–бҹ’бһҡбһ№бһҖ
1. бһ”бһҫбһҖбһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– `admin/search-insights` вҶ’ бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһ‘бһ·бһҹбһ“бҹҗбһҷ (synonyms) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ¶бһҖбҹ’бһҷбһҠбҹӮбһӣбһӮбҹҒбһҡбһҖбһҳбһ·бһ“бһғбһҫбһүбһүбһ№бһҖбһүбһ¶бһ”бҹӢ бһ¬бһ…бһ»бһ…бһ–бҹ’бһҡбһ„бһҫбһҷбһҖбһ“бҹ’бһҸбһҫбһҷ (ignore) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ¶бһҖбҹ’бһҷбһҘбһҸбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбҹ”
2. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ (Inbox) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһҠбһ¶бһҹбҹӢбһҸбһҝбһ“бһ–бһё `healthchecks.io` бһ¬бһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“бһ–бһё `ops_events` (Backup failed) бһ–бһёбһҷбһ”бҹӢбһҳбһ·бһүбҹ” бһ”бһҫбһҳбһ¶бһ“ бһҹбһјбһҳбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ”
3. бһҖбҹ’бһҡбһЎбҹҒбһҖбһҳбһҫбһӣбһ•бҹ’бһ‘бһ¶бҹҶбһ„ `admin/data-quality` бһҖбҹ’бһҡбҹӮбһ„бһӣбҹ„бһҳбһ¶бһ“бһҖбһҫбһ“бһЎбһҫбһ„бһҜбһҖбһҹбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбҹӮбһӣбһҒбһјбһ… (Broken links)бҹ”

### M2. бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ (Weekly) вҖ” бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғбһ…бҹҗбһ“бҹ’бһ‘
1. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ Cron / Zima Snapshot**: бһ…бһјбһӣбһ‘бҹ…бһҳбһҫбһӣ `https://library.ptec.edu.kh/api/health` бһ•бҹ’бһ“бҹӮбһҖбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһҸбһҫ `backupAgeHours` бһҸбһ·бһ…бһҮбһ¶бһ„ бҹЈбҹ  бһҳбҹүбҹ„бһ„бһ¬бһ‘бҹҒ? бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ (ZimaOS owner) бһұбҹ’бһҷбһҶбҹӮбһҖбһҳбһҫбһӣбһҗбһҸ `.last-ok` бһ”бһҫбһӣбһҫбһҹбһ–бһё бҹЁ бһҗбҹ’бһ„бҹғбҹ”
2. **бһўбһ“бһ¶бһҳбҹҗбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ**: бһ…бһјбһӣбһ‘бҹ… Supabase вҶ’ Table Editor вҶ’ `admin_audit_log` бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ•бҹ’бһҸбһӣбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ· (role changes) бһҖбһ¶бһӣбһ–бһёбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҳбһ»бһ“бһҸбһҫбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ¬бһ‘бҹҒбҹ”
3. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҡбһҝбһ„бһҹбҹ’бһ‘бһҪбһ“ (Deduplication)**: бһ…бһјбһӣбһ‘бҹ… `admin/review` (бһҡбҹҖбһ”бһҸбһ¶бһҳ status=published, бһҸбһҳбҹ’бһҡбҹҖбһ”бһҸбһ¶бһҳбһ…бҹҶбһҺбһ„бһҮбһҫбһ„) вҶ’ бһҡбһҖбһҳбһҫбһӣбһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһҹбҹ’бһҡбһҠбҹҖбһ„бҹ—бһӮбҹ’бһ“бһ¶ бһ бһҫбһҷ archive бһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһҹбҹ’бһ‘бһҪбһ“бһ…бҹ„бһӣбҹ”
4. **Email / Contact**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бҹҶбһ–бҹҒбһүбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„ (Contact form) бһ“бҹ…бһӣбһҫбһңбҹҒбһ”бһҹбһ¶бһҷ бһ•бҹ’бһүбһҫбһҳбһҖбһҖбһ¶бһ“бҹӢбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶ Gmail App-Password бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһӮбҹҒбһ”бһ·бһ‘бҹ”

### M3. бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҒбҹӮ (Monthly) вҖ” бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ‘бһёбһҳбһҪбһҷ
1. **бһҹбһңбһ“бһҖбһҳбҹ’бһҳбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ (User Audit)**: бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ· Admin/Librarian бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһӣбһ»бһ”бһҹбһ·бһ‘бҹ’бһ’бһ·бһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһӣбҹӮбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһ¬бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ” (RUNBOOKS В§M17 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҒбһӣбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһҲбһ”бҹӢ)бҹ”
2. **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҖбһјбһҠ (Dependency Update)**: бһҡбһҸбҹӢ `npm outdated` бһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ (Local)бҹ” Update бһҖбһүбҹ’бһ…бһ”бҹӢбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ (Minor/Patch), Run бһҸбҹҒбһҹбҹ’бһҸ, бһҡбһҪбһ… Deployбҹ” бһҖбһ»бҹҶбһ‘бһ¶бһ“бҹӢ Update бһҖбһүбҹ’бһ…бһ”бҹӢбһ’бҹҶбҹ— (Major) бһӣбһ»бҹҮбһҸбҹ’бһҡбһ¶бһҸбҹӮбһҳбһ¶бһ“бһ–бҹҒбһӣбһҸбҹҒбһҹбҹ’бһҸбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбҹ”
3. **бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ (Alert Review)**: бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ Alert-catalog бһҸбһҫбһҳбһ¶бһ“бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҺбһ¶бһҠбҹӮбһӣбһӣбҹ„бһҸбһҒбһ»бһҹбһүбһ№бһҖбһүбһ¶бһ”бҹӢ (Noise) бһҠбҹӮбһҡбһ¬бһ‘бҹҒ? бһ”бһҫбһҳбһ¶бһ“ бһҹбһјбһҳбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһҳбҹ’бһҡбһ·бһҸ (Thresholds)бҹ”
4. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбҹ’бһҸбһ»бһҖ (Storage Check)**: бһ…бһјбһӣбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Supabase бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ‘бҹҶбһ бҹҶ Database бһҗбһ¶бһҸбһҫбһҮбһ·бһҸбһҠбһӣбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһҺбһҸбҹӢ (Limit) бһ бһҫбһҷбһ¬бһ“бҹ…бҹ” бһ”бһҫбһҮбһ·бһҸбһҠбһӣбҹӢ бһҸбҹ’бһҡбһјбһңбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбһӣбҹӢбһ“бһ¶бһҷбһҖбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ (Director) бһҮбһ¶бһҳбһ»бһ“ бҹЈбҹ  бһҗбҹ’бһ„бҹғбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһЎбһҫбһ„ Planбҹ”

### M4. бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҸбҹ’бһҡбһёбһҳбһ¶бһҹ (Quarterly) вҖ” бһҒбҹӮбһҳбһҖбһҡбһ¶, бһҳбҹҒбһҹбһ¶, бһҖбһҖбҹ’бһҖбһҠбһ¶, бһҸбһ»бһӣбһ¶
1. **бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Disaster Recovery Drill)**: бһҡбһҸбҹӢ `node scripts/backup/restore-drill.mjs` бһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Backup бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ”бһ„бҹ’бһўбһҹбҹӢбҹ” бһ…бһҳбҹ’бһӣбһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ `docs/drills/` бһ бһҫбһҷбһҠбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бҹ”
2. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҜбһҖбһҹбһ¶бһҡ Config (Config Fingerprint)**: бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ `backup-config.json` бһ’бҹҖбһ”бһ“бһ№бһ„ Vercel бһ“бһ·бһ„ Cloudflare бһҹбһ–бҹ’бһңбһҗбҹ’бһ„бҹғбҹ” бһ”бһҫбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ (бһ§. бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбһҗбҹҒбһҡбһҗбҹ’бһҳбһё) бһҹбһјбһҳ Update бһҖбҹ’бһ“бһ»бһ„ `ZIMAOS-DEPLOYMENT.md`бҹ”

2. бһҠбһ¶бһҖбҹӢ Key бһҗбҹ’бһҳбһёбһ…бһјбһӣбһ‘бҹ… Vercel бһ“бһ·бһ„ ZimaOS `.env` бһҡбһҪбһ… Restart Appбҹ”
3. бһ”бһҫбһӣбҹҒбһ…бһ’бҹ’бһӣбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„ Git вҶ’ бһҖбһ»бҹҶбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһӣбһ»бһ” (Delete) бһҖбҹ’бһ“бһ»бһ„ Commit бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһ“бҹ…бһҳбһ¶бһ“бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫ `git filter-repo` бһҠбһҫбһҳбҹ’бһ”бһёбһӣбһ»бһ”бһ…бҹҒбһүбһ–бһёбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„ бһҡбһҪбһ… Force Push бһ‘бҹ… GitHub бһңбһ·бһүбҹ” (бһҸбҹӮбһ”бһҫбһ”бһ¶бһ“бһ”бҹ’бһҸбһјбһҡбһӣбҹҒбһҒбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһҪбһ…бһ бһҫбһҷ бһӣбҹҒбһҒбһ…бһ¶бһҹбҹӢбһҖбҹҸбһӣбҹӮбһ„бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһҡ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҖбһ»бҹҶбһ—бҹҗбһҷбһҹбҹ’бһӣбһ“бҹӢбһҹбҹ’бһӣбҹ„бһ–бҹҒбһҖ)бҹ”

### I12. `malware-upload` бһҳбҹҒбһҡбҹ„бһӮ (Sev 2)
**бһҡбҹ„бһӮбһҹбһүбҹ’бһүбһ¶**: бһҖбһ¶бһҡбһҹбҹ’бһҖбҹҒбһ“бһҳбҹҒбһҡбҹ„бһӮбһҡбһ”бһҹбҹӢ App бһӣбҹ„бһҸбһ”бҹ’бһӣбһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһӣ Admin бһ”бһүбҹ’бһ…бһјбһӣбҹ”
**бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ**:
1. бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһҸбһҫбһңбһ¶бһҮбһ¶бһҳбҹҒбһҡбҹ„бһӮбһҳбҹӮбһ“ бһ¬бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһҡбһЎбҹҶ (False positive)бҹ” бһҳбһҫбһӣ Log бһҖбҹ’бһ“бһ»бһ„ `admin_audit_log` бһҗбһ¶бһўбҹ’бһ“бһҖбһҺбһ¶бһҮбһ¶бһўбҹ’бһ“бһҖ Upload?
2. бһ”бһҫбһҮбһ¶бһҳбҹҒбһҡбҹ„бһӮбһҳбҹӮбһ“: бһҜбһҖбһҹбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ¶бһҡбһ¶бҹҶбһ„бһҳбһ·бһ“бһұбҹ’бһҷбһ…бһјбһӣ Storage бһ‘бҹҒ (бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–)бҹ” бһ–бҹ’бһҡбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҠбҹӮбһӣ Upload бһ“бҹ„бҹҮ (бһўбһ¶бһ…бһӮбһ¶бһҸбҹӢбһҶбҹ’бһӣбһ„бһҳбҹҒбһҡбҹ„бһӮбһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһӮбһ¶бһҸбҹӢ)бҹ”
3. бһ”бһҫбһҜбһҖбһҹбһ¶бһҡбһ“бҹ„бҹҮбһҶбҹ’бһӣбһ„бһ…бһјбһӣбһҡбһҪбһ…бһ бһҫбһҷбһ‘бһҫбһ”бһҹбҹ’бһҖбҹҒбһ“бһғбһҫбһү (бһҖбһҳбҹ’бһҡ) вҶ’ бһ…бһјбһӣбһ‘бҹ…бһӣбһ»бһ”бһ…бҹҒбһүбһ–бһё ZimaOS бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹғбһ—бҹ’бһӣбһ¶бһҳ (`rm /zima/data/books/...`)бҹ”

### I13. `rate-limit-storm` / `captcha-storm` / `waf-spike` (DDoS) (Sev 2)
**бһҡбҹ„бһӮбһҹбһүбҹ’бһүбһ¶**: бһҳбһ¶бһ“бһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ¬бһ”бһ¶бһүбҹӢбһҹбҹҶбһҺбһҫ (Requests) бһҡбһ¶бһ”бҹӢбһҳбҹүбһәбһ“бһҠбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ“бһ¶бһ‘бһёбҹ”
**бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ**:
1. бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳ `DDOS-PROTECTION.md` бһ—бҹ’бһӣбһ¶бһҳбҹ”
2. бһ…бһјбһӣ Cloudflare вҶ’ бһ”бһҫбһҖ **Under Attack Mode**бҹ” бһҖбһ¶бһҡбһ“бҹҒбҹҮбһ“бһ№бһ„бһҠбһ¶бһҖбҹӢ Captcha бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһўбҹ’бһ“бһҖбһ…бһјбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶бҹ”
3. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ IP бһ¬бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҺбһ¶бһҠбҹӮбһӣбһ”бһ¶бһүбҹӢбһҳбһҖбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҮбһ¶бһ„бһӮбҹҒ вҶ’ бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹ’бһ”бһ¶бһ”бҹӢ WAF (Firewall rule) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһӣбһ»бһҖ IP/бһ”бҹ’бһҡбһ‘бҹҒбһҹбһ“бҹ„бҹҮбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бҹ”
4. бһ–бҹҒбһӣбһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһҹбҹ’бһ„бһ”бҹӢбһҹбҹ’бһ„бһ¶бһҸбҹӢ (бҹўбһҳбҹүбҹ„бһ„бһҖбҹ’бһҡбҹ„бһҷ) бһ‘бһҫбһ”бһ”бһ·бһ‘ Under Attack Mode бһңбһ·бһүбҹ”

### I17. `backup-failed` / `backup-stale` (Sev 2)
**бһҡбҹ„бһӮбһҹбһүбҹ’бһүбһ¶**: бһҳбһ·бһ“бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһўбҹҠбһёбһҳбҹӮбһӣ Backup бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғ бһ¬ `/api/health` бһӣбҹ„бһҸбһҗбһ¶ Backup бһ…бһ¶бһҹбҹӢбһҮбһ¶бһ„ бҹЈбҹ  бһҳбҹүбҹ„бһ„бҹ”
**бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ**:
1. SSH бһ…бһјбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ (box) бһ бһҫбһҷбһҡбһҸбҹӢ Script бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹғ: `node scripts/backup/backup-db.mjs`бҹ”
2. бһўбһ¶бһ“ Error message: бһҸбһҫбһ–бҹҒбһү Disk? (бһҳбһҫбһӣ В§I7) бһҠбһ¶бһ…бҹӢ Network? бһ¬ Supabase API бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ?
3. бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҸбһ¶бһҳ Error бһ“бҹ„бҹҮбҹ” бһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ·бһ‘бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ (Alert) бһ‘бҹҒ бһҠбһҡбһ¶бһ”бһҺбһ¶ Backup бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҮбҹ„бһӮбһҮбҹҗбһҷбҹ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҒбҹӮбһӣбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”

### M6. бһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҖбһјбһҠ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ»бһ“бһ–бҹҒбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ’бҹҶбһҠбһ»бҹҶ (Pre-migration Backups)
бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Database Migration) бһ¬бһҖбҹӮбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһўбҹ’бһңбһёбһҳбһҪбһҷ бһҸбҹ’бһҡбһјбһңбһҸбҹӮбҹ–
1. **бһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҖбһјбһҠ (Code)**: бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҖбһјбһҠбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһ’бҹ’бһңбһҫбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ Commit бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ Branch бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”
2. **бһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data)**: SSH бһ…бһјбһӣбһ‘бҹ… ZimaOS бһҡбһҪбһ…бһҡбһҸбҹӢ `node scripts/backup/backup-db.mjs` бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ”бҹҶбһ•бһ»бһҸбһ‘бһ»бһҖбҹ” бһҖбһ»бҹҶбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһҸбҹӮбһӣбһҫ Backup бһҖбһ¶бһӣбһ–бһёбһҷбһ”бҹӢбһҳбһ·бһү бһ–бҹ’бһҡбҹ„бҹҮбһўбһ¶бһ…бһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһ…бһјбһӣбҹ”

### M7. бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠ (Deployment Procedure)
1. **Merge Code**: бһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһҠ (Merge Pull Request) бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбһ¶бһҒбһ¶ `main`бҹ”
2. Vercel бһ“бһ№бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶ (Build) бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (Deploy) бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”
3. бһҡбһ„бҹӢбһ…бһ¶бҹҶ Vercel бһӣбҹ„бһҸбһ–бһҺбҹҢбһ”бҹғбһҸбһ„ (Ready)бҹ”
4. бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ Database (Migrations) вҶ’ бһ…бһјбһӣбһ‘бҹ… Supabase SQL Editor бһҡбһҪбһ…бһҷбһҖбһҖбһјбһҠбһ–бһё `supabase/migrations/...` бһ‘бҹ… Run бһҮбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ (бһҳбһҫбһӣбһӣбҹҶбһҠбһ¶бһ”бҹӢбһӣбҹҶбһҠбҹ„бһҷбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„ DEPLOYMENT-PLAN.md)бҹ”
5. бһ…бһјбһӣбһ‘бҹ…бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһңбҹҒбһ”бһҹбһ¶бһҷбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ (Production) бһҸбһҫбһҳбһ»бһҒбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһҠбһҫбһҡбһ¬бһўбһҸбҹӢбҹ”

### M8. бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷбһҖбһјбһҠ (Rollback Procedure)
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһјбһҠбһҗбҹ’бһҳбһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбҹҒбһ”бһҹбһ¶бһҷбһӮбһ¶бҹҶбһ„бҹ–
1. **бһҖбһ»бҹҶбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҮбһҪбһҹбһҮбһ»бһӣ (Fix forward) бһ—бҹ’бһӣбһ¶бһҳбҹ—** бһ”бһҫбһңбһ¶бһҷбһҖбһ–бҹҒбһӣбһӣбһҫбһҹбһ–бһё бҹҘ бһ“бһ¶бһ‘бһёбҹ”
2. бһ…бһјбһӣбһ‘бҹ… Vercel Dashboard вҶ’ бһӮбһҳбҹ’бһҡбҹ„бһ„ (Project) вҶ’ бһ•бҹ’бһ‘бһ¶бҹҶбһ„ Deploymentsбҹ”
3. бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһ¶бһҡ Deploy бһ…бһ¶бһҹбҹӢбһҠбҹӮбһӣбһҠбһҫбһҡбһҹбҹ’бһҡбһҪбһӣ (бһ–бһҺбҹҢбһ”бҹғбһҸбһ„ бһҳбһ»бһ“бһҳбһҪбһҷбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹҒбҹҮ)бҹ”
4. бһ…бһ»бһ…бһҹбһүбҹ’бһүбһ¶бһ…бһ»бһ…бһ”бһё (Three dots) бһҡбһҪбһ…бһҡбһҫбһҹбһҷбһҖ **"Redeploy"** бһ¬ **"Rollback"**бҹ”
5. бһ”бһҫбһҳбһ¶бһ“бһҖбҹӮбһ”бҹ’бһҡбҹӮ Database бһ•бһ„бһҠбҹӮбһҡ бһҸбҹ’бһҡбһјбһңбһҷбһҖбһҖбһјбһҠ Rollback бһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһӣбһҫбһ“бҹғбһҜбһҖбһҹбһ¶бһҡ Migration бһ“бҹ„бҹҮ бһ‘бҹ… Run бһҖбҹ’бһ“бһ»бһ„ Supabase бһңбһ·бһүбҹ”

### M12. бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бһјбһӣбһ”бҹ’бһҡбһҫ (Privilege Access Review)
бһҷбҹ„бһ„бһҸбһ¶бһҳбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ `DATA-GOVERNANCE.md` (бһҡбһ¶бһӣбҹӢбһҸбҹ’бһҡбһёбһҳбһ¶бһҹ):
1. бһ‘бһ¶бһүбһ”бһүбҹ’бһҮбһёбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ· (Staff, Librarian, Admin) бһ…бҹҒбһүбһ–бһё Databaseбҹ”
2. бһҹбһҪбһҡбһ“бһ¶бһҷбһҖбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ (Director) бһҗбһ¶бһўбҹ’бһ“бһҖбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бҹ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһ“бһ·бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ“бҹҒбҹҮбһ‘бҹҖбһҸбһ¬бһ‘бҹҒбҹ”
3. бһ”бһҫбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҲбһ”бҹӢ бһ¬бһӣбҹӮбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ вҶ’ бһ”бһ“бҹ’бһҗбһҷбһҹбһ·бһ‘бҹ’бһ’бһ·бһ‘бҹ…бһҮбһ¶ "Reader" бһңбһ·бһүбҹ”

### M14. бһҖбһ¶бһҡбһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢбһңбһ·бһүбҹ’бһүбһ¶бһ”бһ“бһ”бһҸбҹ’бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (TLS Certificate Expiry)

2. бһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһўбһё (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҒбһ»бһҹ DNS бһҖбһ¶бһҡбһ–бһ¶бһҡ)бҹ”
3. бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ бһўбһ¶бһ…бһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ Renew бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹғ бһ¬бһ‘бһ¶бһҖбҹӢбһ‘бһ„ Cloudflare Supportбҹ”

### M17. бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһ–бҹҒбһӣбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһҗбҹ’бһ“бһ¶бһҖбҹӢ Admin бһҲбһ”бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ (Offboarding)
бһ“бҹ…бһҗбҹ’бһ„бҹғбһҠбҹӮбһӣбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһ“бҹ„бҹҮбһҲбһ”бҹӢ:
1. бһ…бһјбһӣ Supabase вҶ’ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһӮбһҺбһ“бһёбһӮбһ¶бһҸбҹӢ вҶ’ бһ‘бһҳбҹ’бһӣбһ¶бһҖбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ· (Demote) бһ‘бҹ…бһҮбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ“бһ’бһҳбҹ’бһҳбһҸбһ¶ (Reader) бһ¬бһӣбһ»бһ”бһ…бҹҒбһүбһ”бһҫбһӮбһ¶бһҸбҹӢбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбҹ”

3. бһ”бһҫбһӮбһ¶бһҸбҹӢбһҖбһ¶бһ“бҹӢбһӣбҹҒбһҒбһҖбһјбһҠбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ (Keys/Passwords) бһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҺбһ¶бһҳбһҪбһҷ бһҸбҹ’бһҡбһјбһң **бһ”бҹ’бһҸбһјбһҡ (Rotate)** бһӣбҹҒбһҒбһҖбһјбһҠбһ“бҹ„бҹҮбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ—бҹ’бһӣбһ¶бһҳ (Vercel, ZimaOS, Supabase, Cloudflare, Gmail)бҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбҹ”
